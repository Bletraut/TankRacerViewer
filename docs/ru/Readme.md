## 👀 Что это?

**Tank Racer Viewer** - это инструмент, созданный с помощью реверс-инжиниринга (исключительно в образовательных и исследовательских целях). Он предназначен для просмотра ресурсов, хранящихся в файлах `.DAT` (также известных как FASTFILE) из игры *Tank Racer (1999)*.  

Использование максимально простое: запустите программу и выберите папку с установленной игрой Tank Racer (предполагается, что она у вас уже есть 😉). После небольшой загрузки программа обработает файлы и отобразит их содержимое.  

На данный момент поддерживается только просмотр ресурсов - текстуры, модели, уровни, текстовые данные и т.д. Экспорт и перепаковка не реализованы.

### Поддерживаемые возможности:
✅ Просмотр основных ресурсов игры: текстуры, модели, бэкграунды, текстовые файлы, модели танков, уровни  
✅ Рендеринг, приближенный к оригинальной игре: прозрачность, двусторонние полигоны, биллборды  
✅ Возможность выбирать бэкграунд, номер круга, а также скрывать/показывать объекты на уровне  
✅ Просмотр используемых текстур модели и их расположения  

### Известные проблемы:
⚠️ В версии **DesktopGL** на Windows (а также, возможно, на macOS и Linux) при открытии диалогового окна выбора файлов с помощью горячих клавиш (Ctrl+O) теряются события отпускания клавиш. В результате система продолжает считать клавиши зажатыми до следующего нажатия.

## 📂 Структура проекта

Решение разделено на несколько проектов:

`FastFileUnpacker` - ключевой компонент. Небольшая библиотека на чистом C#, отвечающая за распаковку файлов игры *Tank Racer (1999)*. Создана на основе реверс-инжиниринга форматов файлов игры.

`ComposableUi` - слой пользовательского интерфейса. Небольшая библиотека на базе MonoGame, созданная в образовательных и экспериментальных целях. Вдохновлена визуальным стилем Aseprite, а также Unity и Flutter.

`TankRacerViewer.Core` - основная логика и ядро проекта.

`DesktopCommon` - общий код для десктопных платформ.

`TankRacerViewer.WindowsDX` - версия для Windows с использованием DirectX.

`TankRacerViewer.DesktopGL` - кроссплатформенная версия (Windows, macOS, Linux) с использованием OpenGL. Имеет зависимость от NativeFileDialog.

## 🛠️ Используемые технологии

Проект построен на MonoGame и ориентирован на кроссплатформенность. Стабильно работает на Windows (DirectX) и, вероятно, будет работать на macOS и Linux через OpenGL (не тестировалось).

- .NET 8  
- [MonoGame](https://monogame.net/)  
- HLSL (шейдеры)  
- C# (основная логика)
